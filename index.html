A Jornada

Comecei desmontando computadores antes de a maioria das pessoas ter um em casa. Em 1998, transformei essa obsessão em carreira — construindo e reparando máquinas, oferecendo suporte técnico, e aprendendo cada camada da stack, de cima para baixo. Essa base prática moldou como penso sobre tecnologia: prático em primeiro lugar, sempre.

Ao longo de 25+ anos, passei do suporte para empreendedorismo — lançando negócios, gerenciando equipes, e eventualmente transitando para desenvolvimento web e software. Mudar do Brasil para os Estados Unidos adicionou outra dimensão: adaptabilidade, fluidez intercultural, e o impulso de reconstruir do zero mais de uma vez.

Habilidades

Web Development JavaScript / Node.js Infraestrutura IT Hardware & Rede IA & Automação Empreendedorismo Python Linux / WSL

O Que Estou Construindo

Agora estou imerso em tecnologias web modernas — trabalhando com ferramentas IA, pipelines de automação, e desenvolvimento full-stack. Este site é tanto um portfólio quanto um laboratório: tudo aqui é construído, iterado, e entregue por mim.

Filosofia

Acredito que a melhor tech é construída por pessoas que sujaram as mãos. Já consertei placas de circuito, gerenciei negócios, imigrei para um novo país, e aprendi novas línguas — técnicas e humanas. Todo problema é só outro sistema para entender. Não estou caçando tendências; estou construindo coisas que funcionam.
